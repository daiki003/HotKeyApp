# アプリ概要
- このアプリは、Windows上での様々な操作をキーボードのみで素早く行うことを目的としています。
  一部マウスを使う操作や、マウスでも代用できる操作はあってもよいですが、基本的にはキーボードだけで完結するのが望ましいです。
- 操作感はVsCodeを目指しています（Alt + ←で1つ前のページに戻るなど）

# 修正依頼時の対応
- 修正後にテストビルドしてくれる際、基本的にアプリを開いている状態なのでビルドが失敗すると思います。
その際、即アプリを強制終了していいので、ロックされたまま何度も繰り返して時間を無駄にすることはしないでください。

# 作業進行のルール
- 作業を進める前にユーザーへ質問がある場合は、必ず回答を待ってから実装を開始してください。
  自動承認ポリシーで承認されていても、未回答の質問がある場合は待機してください。
- **作業完了後の必須手順**:
  コードの修正やビルド確認が終わった後は、必ず以下の手順を**一連の動作として**完了させてください。
  1. `taskkill /f /im HotKeyCommandApp.exe` で実行中のアプリを強制終了する。
  2. `dotnet build` でビルドを通す。
  3. `dotnet run` でアプリを起動し、ユーザーがすぐに確認できる状態にする。
