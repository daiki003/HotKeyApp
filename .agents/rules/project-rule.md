# アプリ概要
- このアプリは、Windows上での様々な操作をキーボードのみで素早く行うことを目的としています。
  一部マウスを使う操作や、マウスでも代用できる操作はあってもよいですが、基本的にはキーボードだけで完結するのが望ましいです。
- 操作感はVsCodeを目指しています（Alt + ←で1つ前のページに戻るなど）

# 修正依頼時の対応
- 修正後にテストビルドしてくれる際、基本的にアプリを開いている状態なのでビルドが失敗すると思います。
その際、即アプリを強制終了していいので、ロックされたまま何度も繰り返して時間を無駄にすることはしないでください。
